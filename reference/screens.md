   - Отчество
   - Email
   - Телефон

Правая часть
 * Адрес
   - Форма (Radio)
     + Город
     + Улица
     + Дом
     + Корпус
     + Строение
     + Квартира
   - Сохраненный адрес 1 (Radio)
   - Сохраненный адрес 2 (Radio)

Ниже - по центру
